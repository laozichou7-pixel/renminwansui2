人类社会交换形式的三个历史阶段

阶段一：物物交换

W — W

原始公社末期。我用粮食换你的猎物。目的是使用价值——我要你的东西是因为我能用。

阶段二：简单商品流通

W — G — W

小农和手工业者。卖粮得钱，用钱买布。货币只是中间人。

阶段三：资本总公式

G — W — G′

资本主义。货币是目的。G′ = G + ΔG。ΔG = 剩余价值 = 利润的来源。

G — W — G′
资本总公式
G 怎么变成 G′ 的？
那个多出来的 ΔG 是从哪里来的？
在流通领域低买高卖 → 只是财富重新分配
秘密不在流通领域，而在生产领域。

秘密：劳动力商品

资本家的"发现"

市场上存在一种特殊商品——劳动力（A）。

劳动力的特殊之处：它能创造出超过自身价值的价值。

G — W〈A, Pm〉…… P …… W′ — G′

资本家付给工人一天工资（v），工人却创造出比工资更大的价值（v + m）。

资本不是钱，不是机器，不是工厂。
资本是一种社会关系。

W = c + v + m
商品价值构成公式
c = 不变资本（生产资料转移的价值）
v = 可变资本（劳动力再生产所需成本）
m = 剩余价值（劳动者创造的超过劳动力价值的部分）
剩余价值率 m′ = m / v
揭示剥削程度——工人创造的价值中有多少被无偿占有

为什么必然爆发危机？

资本主义的根本矛盾

生产社会化 ↔ 生产资料私人占有

传导机制

① 资本压低劳动者工资（v）→ 压低成本

② 机器替代工人 → 产业后备军 → 更多人失去收入

③ 劳动者购买力增速 < 生产能力扩张速度

结果

一边是堆积如山的商品，
一边是买不起商品的人民。

经济危机
剩余价值实现危机
经济根源：商品卖不掉 → 资本循环断裂 → 企业倒闭 → 工人失业
+
阶级力量重置周期
阶级动态：资产阶级利用危机压低工资、瓦解工会、重置剥削条件
平时吸血 → 危机时输劳动者的血 → 输完继续吸血

破除虚假的二元选择
确立民主计划经济

市场经济
G — W … P … W′ — G′
货币 → 商品（劳动力+生产资料）…… 生产过程 ……
包含剩余价值的商品 → 增殖后的货币

中间的W′被生产出来，不是因为社会需要它，
而是因为它能产生利润（G′ ＞ G）。

市场经济的根本问题

它以利润为生产目的，而不是以人的需要为生产目的。

住房是商品 → 资本制造稀缺抬高房价，不会解决无家可归者

医疗是商品 → 资本把疾病当利润机会，不会根除疾病的社会根源

教育是商品 → 资本把教育变成阶级再生产的筛选器

结论

市场经济不是答案。

但官僚计划经济也不是答案

P …… W′ — G′ · G — W〈A, Pm〉…… P

苏联的教训

劳动者不能查账——不知道G′的真实规模和去向

劳动者不能监督——计划机关不受群众控制

劳动者不能罢免——脱离群众的干部不会被撤换

劳动者不能反馈——需求信息被官僚层级阻隔

核心问题

"计划权脱离群众"——不是计划不行，是计划权不在人民手里。

民主计划经济的八条制度原则

原则在经济计划中的体现

公开透明每一笔剩余产品去向，劳动者可通过数据库实时查阅

Ⅳ

过渡路径
货币如何消亡，计划如何进化

过渡期：计划主导，货币工具化

资本主义的货币

一切商品的一般等价物 → 可自由转化为资本 → 可无限积累生息遗传 → 支配他人劳动的权力工具

过渡期的货币

计划分配的核算符号和劳动凭证 → 禁止自由转化为私人资本 → 消费领域保留，生产领域由计划直接配置

W′ — G′ · Pl — W〈A, Pm〉…… P …… W′

那个 · Pl 就是站在桥头堡的交通警察——
规定货币的流向、速度和分配比例。

货币的逐步替代：信息 + 积分

替代一：用信息替代价格信号

货币→价格→供需→事后调节、盲目、伴随危机

大数据+AI智能计划平台→实时汇总社会需要→直接生成资源配置方案→供需匹配在事前完成

替代二：用积分替代一般购买力

资本主义货币社会主义积分
衡量什么交换价值社会贡献的时间与质量
能买什么一切，包括生产资料绑定用途的个人消费（禁止购买Pm）
能否生息可以，转化为剥削工具定期清零，不能转化为资本
追求目标符号无限累积贡献与权利合一
终极形态：货币消亡，计划内化
N → 智能计划平台
→ 自动化生产 ……
→ 按需分配 → N′
G（货币）消亡了。
社会不再需要一种专门充当一般等价物的中介商品。
计划（Pl）也不表现为官僚机构——
它内化为社会生产和生活的基础设施本身，
就像今天的水电网。

共产主义社会的愿景

一、失业的消灭

劳动力闲置率 → 接近零

不是因为"强制就业"——而是社会生产不以压低工资成本为目的。

当机器替代了人的岗位 → 社会不是把人扔进产业后备军 → 而是缩短所有人的劳动时间。

技术进步带来的自由时间 → 归全体劳动者共享，而非归资本独占利润。

二、剩余价值 → 剩余产品

资本主义下

你创造的超过你工资的价值（m），被资本无偿占有——不知道去了哪里，不知道养肥了谁。

民主计划经济下

你在手机APP上可以看到：
你本月贡献的剩余产品 →
被用于哪个民生工程、帮助了哪个生病的人、
投入了哪个偏远地区的发展。

这不是监控——是透明的公共账本。

三、生产品替代商品

商品

为了交换而生产 → 价值必须在市场上"实现" → 卖不掉就危机

生产品

为了需要而生产 → 不需要"在市场上证明自己的价值" → 被生产出来，分配给需要的人

它失去了商品最核心的属性：价值形式的异化。

资本主义最残忍的剥夺，
不是对工资的剥夺，
而是对时间的剥夺。
今天的生产力水平下，每人每周只需工作10-15小时
就能满足全社会的物质需要。
为什么还要996？

因为资本不能让劳动者闲下来——
一旦有了自由时间，他们就会思考、组织、追问。

不是生产需要，是统治需要。
在共产主义社会高级阶段……
在劳动已经不仅仅是谋生的手段，
而且本身成了生活的第一需要之后……
社会才能在自己的旗帜上写上：
各尽所能，按需分配！
—— 马克思《哥达纲领批判》

四、按需分配的起步：生存资料

不是一步到位——从生存资料开始

住房、医疗、教育、基本食品

这些不是以劳动为前提的"报酬"——是人的基本权利。

今天的生产力已完全具备条件

粮食够养活所有人 → 有人挨饿

空置房几千万套 → 有无家可归者

可治愈的疾病 → 杀死付不起钱的穷人

不是生产力问题，是制度问题。
